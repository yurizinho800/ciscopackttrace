# Projetos de Redes no Cisco Packet Tracer

📡 Este repositório reúne minhas criações durante os estudos de Redes de Computadores.  
💻 Todos os projetos foram feitos no Cisco Packet Tracer.  
🎥 Confira também meu vídeo demonstrando os projetos: [https://youtu.be/v4uvaqSXGEw]

## Exemplos de projetos
- 🌐 Topologia de rede simples com 3 PCs, 1 switch, 3 servidor e 1 roteador
- 📶 Configuração de roteador com DHCP
- 🔒 Rede com VLANs e segurança básica

## Como abrir
1. Baixe o arquivo `.pkt`
2. Abra no Cisco Packet Tracer
